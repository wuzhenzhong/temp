# jQuery 参考手册 - 核心



## jQuery 核心函数

| 函数                                                         | 描述                                                      |
| :----------------------------------------------------------- | :-------------------------------------------------------- |
| [jQuery()](https://www.w3school.com.cn/jquery/core_jquery.asp) | 接受一个字符串，其中包含了用于匹配元素集合的 CSS 选择器。 |
| [jQuery.noConflict()](https://www.w3school.com.cn/jquery/core_noconflict.asp) | 运行这个函数将变量 $ 的控制权让渡给第一个实现它的那个库。 |
