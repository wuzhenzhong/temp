## Node.js JXcore 打包

Node.js是一个开放源代码、跨平台的、用于服务器端和网络应用的运行环境。

JXcore是一个支持多线程的 Node.js 发行版本，基本不需要对你现有的代码做任何改动就可以直接线程安全地以多线程运行。

本文主要介绍JXcore的打包功能。

## JXcore 安装

下载JXcore安装包，然后进行解压，在解压的目录下提供了jx二进制文件命令，接下来我们主要使用这个命令。
