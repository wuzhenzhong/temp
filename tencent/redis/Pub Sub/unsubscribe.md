# unsubscribe

```javascript
UNSUBSCRIBE [channel [channel ...]]
```

**自2.0.0起可用。**

[纠错](javascript:;)

**时间复杂度：** O（N）其中N是已订阅频道的客户端数量。

取消订阅来自给定频道的客户，或者如果没有提供任何频道，则从所有频道取消订阅。

当没有指定频道时，客户端将从所有先前订阅的频道中取消订阅。在这种情况下，每个未订阅频道的消息将发送到客户端。
